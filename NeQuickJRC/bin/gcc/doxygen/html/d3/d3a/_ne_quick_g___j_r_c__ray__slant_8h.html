This function returns electron density at the specified point along a slanted ray. See F.2.8.1.

  • Adjusts position information for current position along ray
  • Recalculates ionosphere information for new latitude and longitude (iono_profile_get)
  • Gets the electron density with a call to electron_density_get
Parameters
[in,out]pContextNeQuick context
[in]height_kmDistance of the point along the ray in km
[out]pElectron_densityMaximum Ne in electrons/m3
