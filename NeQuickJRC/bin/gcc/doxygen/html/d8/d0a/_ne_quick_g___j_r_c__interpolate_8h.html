Third Order Interpolation function, See 2.5.7.1.

Be P1 = (-1,z1), P2 = (0,z2), P3 = (1,z3), P4 = (2,z4). If P = (x,zx), to compute the interpolated value zx at the position x, with x∈[0,1], the following algorithm is applied:

If |2x| < 2NEQUICK_G_JRC_INTERPOL_EPSILON ⇒ zx = z2
else

δ = 2x - 1
g1 = z3 + z2
g2 = z3 - z2
g3 = z4 + z1
g4 = (z4 - z1) / 3
a0 = 9g1 - g3
a1 = 9g2 - g4
a2 = g3 - g1
a3 = g4 - g2
zx = (1/16) * (a0 + a1δ + a2δ2 + a3δ3)
Parameters
[in]interpol_points4 interpolation points: z1, z2, z3 and z4
[in]offsetposition x where x∈[0,1]
Returns
interpolated value zx
