This function calculates electron content at the specified height, in the bottom part of the ionosphere below the F2 peak height. The function sums semi-Epstein Layers with a modification to reduce excessive Ne around F2 peak and 100km.

height_kmThe height at which the electron density is required in km
Returns

Returns the electron density given the current height using the calculated ionospheric profile

See 2.5.6 Electron density computation:
Two different modules have to be used accordingly to the height considered.
In particular:
if h <= hmF2

  • the bottomside electron density has to be computed using bottom_side
    if h < hmF2
  • the topside electron density has to be computed using top_side

[in,out]pProfileionospheric profile
[in]height_kmheight in km
