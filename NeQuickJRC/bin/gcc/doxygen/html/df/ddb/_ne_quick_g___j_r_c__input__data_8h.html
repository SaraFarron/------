Detailed Description

NeQuickG input data for ray and time

In order to compute the slant TEC along a ray path, the NeQuick G model requires information on the properties of that ray path (start and end points), and an indication of the time at which the values are required:

  • φ1 Geodetic latitude from receiver
  • λ1 Geodetic longitude from receiver
  • h1 Geodetic height from receiver
  • φ2 Geodetic latitude from satellite
  • λ2 Geodetic longitude from satellite
  • h2 Geodetic height from satellite
  • UT time
  • Month

See 2.3 Inputs and Outputs
