#define ITU_R_P_371_8_LOWER_SOLAR_FLUX_IN_SFU   (63.7)

The lowest Solar Flux value in average conditions that NeQuick is expected to operate on. It corresponds to a Sun Spot Number of 0. Eq. 17
This value is considered adequate when no other solution is available, while still being able to correct for a significant contribution of the ionospheric delay error. unit is sfu (solar flux unit) where 1 sfu = 10-22 W/(m2*Hz)
